## Grammar Review

### 1) 形容词搭配与介词

- Original: `I'm too shiny for my dress.`
- Better: `I'm too shiny in this dress.`
- If you mean the dress itself: `This dress is too shiny.`
- Note: 说“我穿这条裙子显得很闪”更自然用 `in this dress`。

### 2) 将来时结构

- Wrong: `Tomorrow will raining.`
- Correct: `Tomorrow it will rain.`
- Also natural: `It's going to rain tomorrow.`
- Note: `will` 后接动词原形，不能直接接 `raining`。

### 3) 祈使句否定 + 形容词

- Wrong: `Don't afraid.`
- Correct: `Don't be afraid.`
- Note: `afraid` 是形容词，需要 `be`。

### 4) 完整句与时间从句

- Fragment: `After the rain clears.`
- Correct: `After the rain clears, we will go for a walk.`
- Note: 这是从句，后面要接主句。

### 5) 单词选择：`sun` vs `sign`

- Misheard sentence: `There is a sun behind me.`
- If talking about a board/logo: `There is a sign behind me.`
- If talking about the sky: `The sun is behind me.`

### 6) `some` 后要有名词

- Wrong: `There is some behind me.`
- Correct: `There is a sign behind me.`
- Correct: `There are some signs behind me.`

### 7) `workflow` 拼写

- Standard: `workflow` (one word)
- Natural sentence: `This workflow doesn't exist.`

### 8) 不定式 `to`

- Wrong: `The first way is walk up to the registers.`
- Correct: `The first way is to walk up to the registers.`

### 9) 可数名词复数

- Wrong: `You can order stuff like a smoothies.`
- Correct: `You can order stuff like smoothies.`

## Pronunciation Tips

- `sign` /saɪn/ vs `sun` /sʌn/: `sign` 是长双元音 /aɪ/，`sun` 是短元音 /ʌ/。
- `charcoal` /ˈtʃɑːrkoʊl/ vs `chuckle` /ˈtʃʌkəl/: 中间元音和尾音都不同，避免混读。
- `skinny` /ˈskɪni/ vs `shiny` /ˈʃaɪni/: 首辅音 /sk/ 与 /ʃ/、中间元音都不同。
- `treats` /triːts/: 结尾 `-ts` 要读出来，不要吞音。
- `workflow` /ˈwɜːrkfloʊ/: 重音在第一音节 `work`。
