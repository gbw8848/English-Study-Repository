## 🔧 语法复盘 (Grammar Review)

### 1. "What should I do next?" (问"做什么")
- ✅ 推荐表达: What should I do next?
- ✅ 替换说法: What do I do next? / How should I proceed?
- 💡 解析: do 后面通常需要一个"做什么"的内容,用 what 来充当宾语更自然。

### 2. "Which one do you recommend?" (recommend 后不重复宾语)
- ✅ 推荐表达: Which one do you recommend?
- ✅ 扩展说法: Which one do you recommend I use?
- 💡 解析: which one 已经是宾语,再加 it 会重复。

### 3. "what caused it" (what 作主语)
- ✅ 推荐表达: Please tell me what caused it.
- ✅ 更口语: Can you tell me what's causing this?
- 💡 解析: what caused it = "是什么导致了它",结构完整。

### 4. Google Sheet(s) 单复数
- ✅ 单数(一个表): My Google Sheet hasn't changed.
- ✅ 复数(多个表): My Google Sheets haven't changed.
- 💡 解析: 单数用 has/hasn't,复数用 have/haven't。

### 5. If 条件句表达
- ✅ 推荐表达: If it works, set the status to "Success".
- 💡 解析: If + 条件,后面接动作; status 是更明确的宾语。

### 6. 疑问句 vs 宾语从句
- ✅ 直接提问: Where is the problem?
- ✅ 放在句子里: Can you check where the problem is?
- 💡 解析: 单独提问要用疑问句语序; where the problem is 通常用作从句。

## 🗣️ 精选短语 (Useful Phrases)

1. **How do I fill in the fields when creating an API key?** (创建 API key 时这些字段怎么填?)
2. **I've already created the API key.** (我已经创建好 API key 了。)
3. **The workflow is working now, but my Google Sheet isn't changing.** (工作流能跑了,但表格没变化。)
4. **For example, the status isn't updating.** (比如 status 状态没有更新。)
5. **Can you check where the problem is?** (你能帮我看看问题在哪吗?)
6. **Here's the detailed error message.** (这是详细的报错信息。)

## 👂 发音强化 (Pronunciation Tips)

### 重点发音练习
- **status** /ˈsteɪ.təs/: 重音在前,读作 "STAY-təs"。
- **detailed** /ˈdiː.teɪld/ 或 /dɪˈteɪld/: 两种发音都常见。
- **workflow** /ˈwɜːrk.floʊ/: work + flow,注意 /wɜːrk/ 的卷舌/儿化感。
- **API** /ˌeɪ.piːˈaɪ/: 每个字母分开读。
- **recommend** /ˌrek.əˈmend/: 重音在最后音节。
