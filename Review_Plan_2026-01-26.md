# Review Plan 2026-01-26

## 核心词汇
- API key /ˌeɪ.piːˈaɪ kiː/: API 密钥。
  - 例句: I’ve already created the API key.
- fill out /fɪl aʊt/: 填写（表格/字段）。
  - 例句: How do I fill out the Create API Key form?
- field /fiːld/: 字段。
  - 例句: How should I fill in the fields when creating an API key?
- workflow /ˈwɜːrk.floʊ/: 工作流程。
  - 例句: The workflow is working now.
- status /ˈsteɪ.təs/: 状态。
  - 例句: The status isn’t updating.
- error /ˈerər/: 错误。
  - 例句: Now it’s giving an error.
- detailed /ˈdiː.teɪld/: 详细的。
  - 例句: Here’s the detailed error message.
- recommend /ˌrek.əˈmend/: 推荐。
  - 例句: Which one do you recommend?

## 语法复盘
1. 错误: How should I do next?
   正确: What should I do next?
   说明: do 需要宾语，what 可以充当“做什么”的宾语；how 只能表示方式，需要搭配宾语才能用。
2. 错误: Which one do you recommend it?
   正确: Which one do you recommend?
   说明: which one 已经是宾语，后面再加 it 会重复。
3. 错误: Please tell me why caused it?
   正确: Please tell me what caused it.
   说明: why 不能作主语；what caused it 才是“是什么导致了它”的结构。
4. 错误: Now the workflow is working. But the Google sites haven’t change.
   正确: Now the workflow is working, but my Google Sheet hasn’t changed.
   说明: Google Sheets 才是正确名称；单数用 hasn’t changed。
5. 错误: If it works and change it to success.
   正确: If it works, set the status to “Success”.
   说明: 条件句用 if + 主句；change 需要明确宾语。
6. 错误: Where the problem is.
   正确: Where is the problem?
   说明: 单独提问要用倒装疑问句；where the problem is 只能做从句。

## 精选短语
- How do I fill in the fields when creating an API key?
- I’ve already created the API key.
- The workflow is working now, but my Google Sheet isn’t changing. For example, the status isn’t updating.
- Can you check where the problem is?
- Here’s the detailed error message.

## 发音强化
- status /ˈsteɪ.təs/: 重音在前，读作 “STAY-təs”。
- detailed /ˈdiː.teɪld/ 或 /dɪˈteɪld/: 两种发音都常见。
- workflow /ˈwɜːrk.floʊ/: wɜːrk 连读要清楚。
- API /ˌeɪ.piːˈaɪ/: 每个字母分开读。
- recommend /ˌrek.əˈmend/: 重音在最后音节。
